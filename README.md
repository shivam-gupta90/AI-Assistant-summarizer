# AI Personal Assistant using Flask and Groq API

The **AI Personal Assistant** is a web-based application developed using **Python Flask**, **HTML**, **CSS**, **JavaScript**, and the **Groq API**. The primary objective of this project is to demonstrate the integration of modern Large Language Models (LLMs) into a web application to perform intelligent text-processing tasks. The application provides users with a simple, interactive interface where they can communicate with an AI assistant and summarize lengthy emails instantly.

The project consists of two core functionalities. The first feature, **Ask Anything**, enables users to ask general questions and receive AI-generated responses in real time. User queries are submitted through a web form and processed by the Flask backend. The backend securely communicates with the Groq API using the OpenAI-compatible Python SDK and retrieves accurate, context-aware responses from the **Llama 3.3 70B Versatile** language model. These responses are returned to the frontend in JSON format and displayed dynamically without reloading the page, creating a smooth and responsive user experience.

The second feature, **Email Summarization**, allows users to paste long email content into a text area and receive a concise summary in two to three sentences. This functionality demonstrates the practical use of AI for improving productivity by helping users quickly understand lengthy messages. The summarization process is handled through carefully designed prompts that instruct the language model to generate clear and informative summaries.

The project follows a modular architecture where the frontend handles user interaction while the Flask backend manages request processing, API communication, and response generation. Sensitive information, such as the API key, is stored securely in a **`.env`** file using the **python-dotenv** library, ensuring better security practices. Asynchronous JavaScript (`fetch`) is used to send requests and update the webpage dynamically without requiring full-page refreshes.

Overall, this project highlights the practical implementation of Generative AI in web development. It demonstrates concepts such as RESTful API integration, prompt engineering, asynchronous communication, environment variable management, and client-server architecture. The application is lightweight, user-friendly, and easily extensible, making it a strong foundation for future enhancements such as chat history, document summarization, multilingual support, voice interaction, authentication, and database integration. This project reflects modern AI-powered application development practices and showcases the ability to build intelligent, scalable, and interactive web solutions using contemporary technologies.
